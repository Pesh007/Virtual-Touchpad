# Virtual-Touchpad
Control your mouse while you're lying in bed

#Description 
Control the cursor by moving your hand while showing one with your index finger. Click by pinching the tips of the middle finger and the thumb. Scroll vertically and horizontally by showing two with your index and middle fingers.

#Requaremnets 
You must install install the 'hand_landmarker.task' from https://ai.google.dev/edge/mediapipe/solutions/vision/hand_landmarker and install the packeges from requarements.txt
